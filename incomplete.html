• Playstation 1 game engine

Incomplete 2D game engine for the Sony Playstation 1 game console. Contains features such as reading files from disc, reading from and writing to the memory card and checking controller status. PSYQ toolkit is required for compilation and it can be unofficially obtained from here. I actually made a more complete PS1 engine years before this one, which was able to do 2D rendering of sprites of various image depths and had audio playback capabilities. However it was not as easy to use as this one and some of the features only worked on the emulator but not the real console. For that reason I decided to omit it, however I'm willing to show its source code to anyone who might be interested.

• Grand strategy game

Game inspired by Europa Universalis IV and Crusader Kings II. The scope of the aforementioned games is gigantic, and hence I knew that I wouldn't be able to make a game of such calibre, so I approached this project without the intention of ever completing it. The map, biomes and provinces are generated procedurally (using Perlin noise and Voronoi maps). Province, country and monarch names are generated as a mashup of pre-existing name bases. Simple concepts such as ethnicity and religion were implemented fully, with partially completed implementations of parliament. The source code was lost.
